# Computer Vision Project

A Street View House Numbers (SVHN) Recognition Project

## Dependencies
```
1- OpenCV
2- json
3- typing
4- collections
5- os
6- cv2
7- numpy
```

## How to Run
 ```
 
1- install the above dependencies

2- open PyCharm

3- Run the code
```
